# 🎨 Portfolio

Um website de portfólio moderno e responsivo, criado com [Lovable](https://lovable.dev/) - uma plataforma de desenvolvimento web baseada em IA.

Construído com **TanStack Start**, **React**, **TypeScript** e **Tailwind CSS**.

## 🎯 Destaques

- ✨ Design responsivo e moderno
- 🚀 Performance otimizada
- 📱 Totalmente mobile-friendly
- 🎨 Componentes customizados com Shadcn/UI
- 🔄 Roteamento avançado com TanStack Router
- 🌐 Deploy na Cloudflare

## 🚀 Tecnologias

- **[TanStack Start](https://tanstack.com/start)** - Framework full-stack
- **[React 19](https://react.dev/)** - UI library
- **[TypeScript](https://www.typescriptlang.org/)** - Type safety
- **[Tailwind CSS 4](https://tailwindcss.com/)** - Styling
- **[TanStack Router](https://tanstack.com/router/)** - Routing
- **[Shadcn/ui](https://ui.shadcn.com/)** - Componentes de UI
- **[Vite](https://vitejs.dev/)** - Build tool
- **[Cloudflare](https://www.cloudflare.com/)** - Deployment
- **[Lovable](https://lovable.dev/)** - AI Web Development

## 📦 Instalação

```bash
# Clonar o repositório
git clone https://github.com/gisellehoekveldsilva/portfolio.git
cd portfolio

# Instalar dependências
npm install
```

## 🏃 Desenvolvimento

```bash
npm run dev
```

O servidor iniciará em `http://localhost:5173`

## 🔨 Build para Produção

```bash
npm run build
```

## 👀 Preview da Build

```bash
npm run preview
```

## 🎨 Lint e Formatação

```bash
# Verificar linting
npm run lint

# Formatar código
npm run format
```

## 📁 Estrutura do Projeto

```
src/
├── components/        # Componentes React
│   ├── ui/           # Componentes Shadcn/UI
│   ├── ProjectTimelineCard.tsx
│   └── SiteHeader.tsx
├── routes/           # Páginas do router TanStack
├── hooks/            # Custom React hooks
├── lib/              # Utilidades e helpers
├── assets/           # Imagens e arquivos estáticos
└── styles.css        # Estilos globais
```

## 🚀 Deploy

Este projeto está configurado para deploy na **Cloudflare** usando o Wrangler.

```bash
npm run build
```
